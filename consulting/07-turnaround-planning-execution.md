# Turnaround planning & execution toolkit

**Use:** to bring discipline to a plant turnaround / shutdown (TAR) — through
planning, execution, and close-out. Turnarounds overrun on scope creep, weak
front-end planning, and poor daily control. This toolkit attacks those three.

**Phase gates:** Initiate → Scope freeze → Detailed planning → Readiness →
Execution → Close-out. Don't pass a gate with the previous one's must-haves open.

- [ ] Target dates, duration, and budget envelope set — `[ ]`

## Phase 2 — Scope development & FREEZE
- [ ] Master work list built from all sources (inspection, maintenance, projects, defects) — `[ ]`
- [ ] Each item challenged: must it be done now, in the TAR, with the plant down? — `[ ]`
- [ ] Scope ranked (mandatory / high-value / discretionary) — `[ ]`
- [ ] **Scope freeze date set and enforced** — `[ ]`
- [ ] Post-freeze change process defined (who approves additions, and the bar) — `[ ]`
- [ ] Growth/discovery allowance agreed (realistic, not zero) — `[ ]`

## Phase 3 — Detailed planning
- [ ] Each work order planned: steps, durations, crafts, hours — `[ ]`
- [ ] Materials, spares, and long-lead items identified and ordered — `[ ]`
- [ ] Long-lead status tracked to delivery (the classic schedule-killer) — `[ ]`
- [ ] Contractor scopes, numbers, and mobilisation planned — `[ ]`
- [ ] Integrated schedule built; **critical path identified** — `[ ]`
- [ ] Resource histogram levelled (no impossible peaks) — `[ ]`
- [ ] Logistics: cranes, scaffolding, access, lay-down, waste — `[ ]`
- [ ] Shift pattern and coverage planned — `[ ]`

## Phase 4 — Readiness (pre-shutdown)
- [ ] Permits-to-work, isolations (LOTO), and energy-isolation plans ready — `[ ]`
- [ ] Safety plan, JSAs, and contractor inductions complete — `[ ]`
- [ ] Materials/spares received, inspected, kitted, and staged — `[ ]`
- [ ] Contractors mobilised, inducted, and briefed on their scope — `[ ]`
- [ ] Daily control structure agreed (meeting cadence, KPIs, war room) — `[ ]`
- [ ] Communication plan and progress-reporting method live — `[ ]`
- [ ] Start-up / re-commissioning plan written — `[ ]`
- [ ] **Readiness review passed; go/no-go decision recorded** — `[ ]`

## Phase 5 — Execution control
- [ ] Daily plan vs. actual on the critical path tracked — `[ ]`
- [ ] Progress measured objectively (% complete by earned hours, not opinion) — `[ ]`
- [ ] Emergent/discovery work logged, assessed, and approved against the bar — `[ ]`
- [ ] Schedule re-forecast daily; recovery plans for slips — `[ ]`
- [ ] Safety performance and permit compliance reviewed daily — `[ ]`
- [ ] Quality holds and inspection sign-offs not skipped under time pressure — `[ ]`
- [ ] Bottlenecks (cranes, permits, labour, access) actively managed — `[ ]`

## Phase 6 — Start-up & close-out
- [ ] Pre-startup safety review / loop checks complete — `[ ]`
- [ ] Phased, controlled start-up per plan — `[ ]`
- [ ] Punch list captured, prioritised, and owned — `[ ]`
- [ ] Demobilisation and site reinstatement done — `[ ]`
- [ ] Cost and schedule actuals reconciled vs. plan — `[ ]`
- [ ] Lessons learned captured while fresh; carried into next TAR — `[ ]`
- [ ] As-builts, CMMS, and asset records updated — `[ ]`
